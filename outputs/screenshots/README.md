# GuidaPlate — Screenshots

Screenshots of the GuidaPlate React frontend.

## How to capture
1. Run: cd frontend && npm run dev
2. Open http://localhost:5173
3. Take screenshots of each page and save here

## Files
| File | Page | What to capture |
|---|---|---|
| 01_dashboard.png | Dashboard | Full dashboard with ML cards and metric stats |
| 02_food_explorer.png | Food Explorer | Food selected, detail panel open, radar chart visible |
| 03_risk_assessment.png | Meal Assessment | Stage selected, foods added, before running |
| 04_risk_result.png | Meal Assessment | HIGH or MODERATE result with nutrient breakdown |
| 05_daily_tracking.png | Meal Assessment | Two meals logged, daily totals visible |
