# NHANES 2017-2018 Data Files

Download these six files from:
https://wwwn.cdc.gov/nchs/nhanes/search/datapage.aspx?Component=Dietary&CycleBeginYear=2017

Required files:
- DR1TOT_J.XPT  (Dietary Day 1 Total Nutrients)
- DR2TOT_J.XPT  (Dietary Day 2 Total Nutrients)
- DR1IFF_J.XPT  (Dietary Day 1 Individual Foods)
- DR2IFF_J.XPT  (Dietary Day 2 Individual Foods)
- BIOPRO_J.XPT  (Laboratory Biochemistry)
- DEMO_J.XPT    (Demographics)

Place all files in this folder.
Do not commit these files to GitHub.
They are in .gitignore.
